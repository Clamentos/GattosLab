FabRISC

FabRISC is an open-source, modular, RISC-like, privileged instruction set architecture for high performance 32 and 64-bit microprocessors. It has floating-point and vector capabilities, as well as atomic and transactional memory for multicore systems.

Logi Block

A small, open-source, FPGA inspired, logic cell implemented with discrete SMD components. It is composed of a fracturable 4-input LUT and a D-type flip-flop. It can perform any 4-to-1 or 3-to-2 boolean function and it can optionally store the result in its flip-flop. Logi Block is programmable serially and can be combined with others to implement any logic circuit.
