Start time: specify the start time of the window. On page load, it's automatically set to the current day. Clearing this field will cause the search to ignore this field and fetch arbitrarily far into the past.

End time: specify the end time of the window. Clearing this field will cause the search to ignore this filter and fetch arbitrarily far into the future.

Paths: specify the request paths to be included in the search. To specify multiple, simply separate the paths with a comma (no spaces). Clearing this field will cause the search to ignore this filter.

HTTP Statuses: specify the response statuses to be included in the search. To specify multiple, simply separate the values with a comma (no spaces). Clearing this field will cause the search to ignore this filter.

Resolution: specify the resolution of the charts in seconds. Clearing this field will cause the charts to default to 10 minutes.
